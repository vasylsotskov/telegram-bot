# Telegram bot example authorization

## Descripton
Some companies need a telegram bot for internal use or for existing customers</br>

<ol>
    <li>Clone this project</li>
    <li>Register in <a>https://telegram.org</a></li>
    <li>In order to get a key for your bot, write to this user https://t.me/BotFather command to create a new bot /newbot</li>
    <li>Rename the file in the project
"default.env" to ".env"</li>
    <li>If you don't know your id send message to https://t.me/chatid_echo_bot</li>
    <li>Create a virtual environment with <code>python -m virtualenv venv</code></li>
    <li>Run source venv/bin/activate to activate the venv</li>
    <li><code>pip install -r requirements.txt</code></li>
    <li>Run bot with python bot.py</li>
    <li>Enjoy!</li>
</ol>
